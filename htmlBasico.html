Html

Html é uma linguagem de marcação usada para definir a estrutura do seu conteúdo. Usa determinados elementos para mostrar seu conteudo, como por exemplo: paragráfos, imagens, links, etc .... São tags que podem ser como essas (<p>conteúdo<\p>) para criar um páragrafo! Os elementos também podem ter atributos que parecem assim: (<a class="link-red">Meu github <\a>). Atributos contém informação extra sobre determinado elemento que você não quer que apareça no conteúdo real. Nesse elemento anterior que vai para meu link do github a class é o (nome) do atrbuto e link-green o (valor) do atributo

CSS

Css é o código que vc usa para dar estilo a sua página web, permite aplicar estililos seletivamente em elementos em documentos html. Por exemplo, selecionar os elementos (<h3>conteúdo<\h3>) da página e torná-los azul.

Conjunto de regras css:

  • h3{ color: blue};
  • Seletor (Selector) O nome do elemento HTML no começo do conjunto de regras. Ele seleciona o(s) elemento(s) a serem estilizados (nesse caso, elementos (<h3>conteúdo<\h3>). Para dar estilo a um outro elemento, é só mudar o seletor.
  • Declaração (Declaration) Uma regra simples como color: blue; especificando quais das propriedades do elemento você quer estilizar.
  • Propriedades (Property) Forma pela qual você estiliza um elemento HTML. (Nesse caso, color é uma propriedade dos elementos <h3>.Em CSS, você escolhe quais propriedades você deseja afetar com sua regra.
  • Valor da propriedade (Property value) À direita da propriedade, depois dos dois pontos, nós temos o valor de propriedade, que escolhe uma dentre muitas aparências possíveis para uma determinada propriedade

Java Script

JavaScript é uma linguagem de programação que adiciona interatividade ao seu site. Isso acontece em jogos, no comportamento das respostas quando botões são pressionados ou com entrada de dados em formulários; com estilo dinâmico; com animação, etc

Funçoes

Functions são uma forma de empacotar a funcionalidade que você deseja reutilizar. É possível definir um corpo de código como uma função que é executada quando você chama o nome da função em seu código. Esta é uma boa alternativa para escrever repetidamente o mesmo código. Você já viu alguns usos de funções. Por exemplo uma função que faz uma multiplicação:

function multiplicacao(num1, num2){

let result = num1 * num2;

return result }

Eventos

A interatividade real em um site requer manipuladores de eventos. Essas são estruturas de código que detectam atividades no navegador e executam o código em resposta. O exemplo mais óbvio é lidar com o evento de click, que é acionado pelo navegador quando você clica em algo com o mouse.
